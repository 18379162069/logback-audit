Project Mailing Lists

A mailing list is an electronic discussion forum that anyone can subscribe to. When someone sends an email message to the mailing list, a copy of that message is broadcast to everyone who is subscribed to that mailing list. Mailing lists provide a simple and effective communication mechanism. With potentially hundreds of subscribers, there is a common set of etiquette guidelines that you should observe. Please keep on reading.

Respect the mailing list type

The "Announce" list is reserved for logback-audit releated annoucements. Please do not post on this list.

The "Support" list is where you can send questions and comments about configuration, setup, usage and other questions about using logback-audit.

Do not post in HTML

Please do your best to ensure that you are not sending HTML or "Stylelized" email to the list. If you are using Outlook or Outlook Express or Eudora, chances are that you are sending HTML email by default. There is usually a setting that will allow you to send "Plain Text" email.
